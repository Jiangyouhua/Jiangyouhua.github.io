* JPart API及应用说明 *

六、Web类说明

1. 说明

Web类是JPart的驱动类，接供必要的接口将HTML页面内容角析为JPart组，并输出在当前位置内


  • Part API

    • 1.Width, Height属性

      当前窗口的宽、高值

    • 2.Alert(string, duration))

      内置的提示条，可定时隐藏

    • 3.AJAX get与post: Get(url, callBack, isJson), Post(url, data, callBack, isJson)

      封装AJAX请求类

    • 4.全局标签：Mark()

      返回全局ID标签

    • 5.从字符串获取对象：ObjectByString(string, split1, split2)

      将key:value;key1:value1或key=value&key1=value1，转对对象

    • 6.解析页面内容为组件，并输出在当前位置内：HTMLParse(selector)

      selector可以为Document.Element对象，解析其内的HTML文档为JPart，并输出当前设位置内

      • 1. data-source=string

        该标签为HTML内容引用JPart的主要标志，根据以下条件获取数据，为JPart提供data

        • 1.1 值为$value, 表示标签将引为一个js变量（如：var value="Jiangyouhua"）
        • 1.2 值为@url, 表示标签将引为一个js的AJAX获取数据（如：Web.Get(url,function(data){})）
        • 1.3 值为Json字符串, 表示标签将解析该Json字符串为数据（如：'{"name":"Jiangyouhua"}'）

      • 2. data-config=string

        该标签获取数据的方式与data-source相同，只是不支持AJAX数据请求，为JPart提供config

      • 3. data-func=string

        处理data-source标签获取的数据，以适应组件的要求

      • 4. data-back=string

        JPart 输出的内容

      • 5. data-part=string

        JPart 输出的内容
