六、HTML类说明

1. 说明

HTML为js用常规类型方式实现构建HTML节点，即: new HTML(), 得到HTML document节点实例

  • 新建一个节点: new HTML(tag, content, attrs...)

    • div : new HTML("div","我是一个DIV对象")

    • a : new HTML("a","我是一个a对象","onclick=alert('Is a!')", "href=#")

    • input : new HTML("input","","type=text value=我是一个input对象")

    • input : new HTML("textarea","我是一个textarea对象")

  • HTML API

    • 1.设置标签：SetTag(name)

      设置标签名称

    • 2.设置、添加属性：SetAttr(key, vlaue), AddAttr(key,value)

      设置属性，会清除已有属性，添加属性则保留已有属性

    • 3.设置、添加样式类型：SetClass(vlaue...), AddClass(value...)

      设置风格类型，会清除已有风格类型，添加风格类型则保留已有风格类型, 可同时添加多个

    • 4.设置、添加样式风格：SetCss(key, value), AddCss(key, value)

      设置样式风格，会清除已有样式风格，添加样式风格则保留已有样式风格

    • 5.设置、添加内容：SetContent(vlaue...), AddContent(value...)

      设置内容，会清除已有内容，添加内容则保留已有内容，可同时添加多个

    • 6.转为Document.Element对象：Element()

      将HTML对象转为Document.Element对象

    • 7.转为html字符串：String()

      将HTML对象转为转为html字符串
